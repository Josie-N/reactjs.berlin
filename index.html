ReactJS.berlin was an early ReactJS usergroup run by @bitcrowd. We at bitcrowd are happy that the React community in berlin has now spawned so many usergroups and events that it does not need this regular meetup as support any longer.

This ReactJS meetup group has been closed

When we started this meetup group in 2015, it was the first ReactJS focused meetup in Berlin and the comunity was still young. Now, some years and many great meetups later, several other meetups, courses and events around React have evolved. So we find that running just another one does not make sense.

Looking for a React meetup, event or course?

If you are looking for a React meetup, event or course, we suggest to have a look at the meetup.com groups. Especially berlinjs.org is a great starting point.

Need a location for your event?

The @bitcrowd office has an event space that you can use for community events. It has a capacity of 90 people (seated) or 30 people (with desks for courses).

Need help from React experts?

The people at @bitcrowd have created many ReactJS applications for their clients over the past years. Just drop us and e-mail or call us and we see how we can help your project.

Code of Conduct
